# No Black Box: Full Visibility & Control

**The agent operates with complete transparency, showing you every file read and every proposed diff.**

Understand exactly what the agent is doing and why — no obfuscation. Review all actions and approve changes before they're made. Checkpoints let you revert easily, keeping you in full control of your codebase. Every gateway request goes through a single InferAll endpoint that can be logged and audited (see the `inferall.auditLog.enabled` setting).
